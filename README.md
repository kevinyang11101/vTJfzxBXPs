# 旅游推荐管理系统简介 java9

## 项目概述

旅游推荐管理系统是基于Spring Boot和Vue.js开发的，集成了协同过滤算法与人脸识别登录功能，致力于为用户提供个性化旅游推荐服务。

## 技术栈

### 系统架构

- B/S架构，前后端分离设计模式。

### 后端技术

- 核心框架：Spring Boot
- 持久层框架：Mybatis/Mybatis-Plus
- 缓存数据库：Redis
- 数据库：MySQL 5.7 / 8.0
- 语言与环境：Java JDK 1.8, Maven IDEA（开发工具）

### 前端技术

- 前端框架：Vue.js
- 开发模式：前后端完全分离，通过API接口与后端交互。

## 功能模块

### 用户认证模块

- 用户注册、登录与登出
- 忘记密码

### 用户前台门户

- 首页：轮播图广告、热门推荐展示（景点、酒店、线路）
- 信息浏览：景点信息、旅游路线、景区酒店详情展示
- 核心交互：景点预约、酒店预订、旅游资讯阅读

#### 个性化功能

- 我的收藏：收藏感兴趣的景点、酒店、线路
- 我的预定：查看和管理个人的所有预约订单

#### 核心亮点

- 智能推荐：基于用户行为和偏好，利用协同过滤推荐算法，为用户生成个性化的推荐列表。
- 人脸识别登录：用户可以使用密码或人脸登录，采用活体检测技术。

### 系统后台管理

- 数据概览：预约数据统计图表（如柱状图、饼图），直观掌握经营状况
- 内容管理：
  - 景点信息管理（增删改查）
  - 旅游路线管理
  - 景区酒店管理
  - 旅游资讯（文章）管理
  - 轮播图管理
- 订单管理：景点预约订单管理、酒店预订订单管理（审核、查看）

## 运行环境

- 前端：现代浏览器环境
- 后端：Java JDK 1.8及以上版本，兼容MySQL 5.7 / 8.0数据库
